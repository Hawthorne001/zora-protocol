# Coins Queries

## API Queries

### Getting a Specific Coin

To fetch detailed information about a specific coin:

```typescript
import { getCoin } from "@zoralabs/coins-sdk";

async function fetchCoin() {
  try {
    const response = await getCoin({
      address: "0xCoinContractAddress",
      chain: 8453, // Optional: defaults to Base (8453)
    });
    
    const coin = response.zora20Token;
    
    console.log("Coin name:", coin?.name);
    console.log("Coin symbol:", coin?.symbol);
    console.log("Total supply:", coin?.totalSupply);
    console.log("Market cap:", coin?.marketCap);
    console.log("24h volume:", coin?.volume24h);
    
    return response;
  } catch (error) {
    console.error("Error fetching coin:", error);
    throw error;
  }
}
```

### Getting Multiple Coins

To fetch information about multiple coins at once:

```typescript
import { getCoins } from "@zoralabs/coins-sdk";

async function fetchMultipleCoins() {
  try {
    const response = await getCoins({
      coinAddresses: [
        "0xFirstCoinAddress",
        "0xSecondCoinAddress",
        "0xThirdCoinAddress"
      ],
      chainId: 8453, // Optional: defaults to Base (8453)
    });
    
    response.zora20Tokens?.forEach((coin, index) => {
      console.log(`Coin ${index + 1}:`);
      console.log(`- Name: ${coin.name}`);
      console.log(`- Symbol: ${coin.symbol}`);
      console.log(`- Market Cap: ${coin.marketCap}`);
    });
    
    return response;
  } catch (error) {
    console.error("Error fetching multiple coins:", error);
    throw error;
  }
}
```

### Getting Coin Comments

To fetch comments for a specific coin:

```typescript
import { getCoinComments } from "@zoralabs/coins-sdk";

async function fetchCoinComments() {
  try {
    const response = await getCoinComments({
      address: "0xCoinContractAddress",
      chain: 8453, // Optional: defaults to Base (8453)
      after: "cursor-value", // Optional: for pagination
      count: 20, // Optional: number of comments to return
    });
    
    response.comments?.forEach((comment, index) => {
      console.log(`Comment ${index + 1}:`);
      console.log(`- Text: ${comment.text}`);
      console.log(`- Created at: ${comment.createdAt}`);
      console.log(`- Author: ${comment.author?.handle || comment.author?.address}`);
    });
    
    // For pagination
    console.log("Next cursor:", response.pagination?.cursor);
    
    return response;
  } catch (error) {
    console.error("Error fetching coin comments:", error);
    throw error;
  }
}
```

## Pagination

Many API queries support pagination through `after` and `count` parameters. Here's how to implement pagination:

```typescript
import { getCoinComments } from "@zoralabs/coins-sdk";

async function paginateThroughComments() {
  let cursor = null;
  const allComments = [];
  const pageSize = 20;
  
  do {
    const response = await getCoinComments({
      address: "0xCoinContractAddress",
      count: pageSize,
      after: cursor,
    });
    
    // Add comments to our collection
    if (response.comments) {
      allComments.push(...response.comments);
    }
    
    // Update cursor for next page
    cursor = response.pagination?.cursor;
    
    // Break if no more pages
    if (!cursor) break;
    
  } while (true);
  
  console.log(`Fetched ${allComments.length} total comments`);
  return allComments;
}
```

## Best Practices

1. **Use API Queries When Possible**: API queries provide more information and are more efficient than onchain queries.
2. **Implement Pagination**: For queries that return potentially large datasets, use pagination to avoid timeouts.
3. **Handle Errors**: Always implement proper error handling for API queries.
4. **Cache Results**: Consider caching query results to reduce API calls and improve user experience.
5. **API Key Security**: Keep your API key secure and don't expose it in client-side code.
